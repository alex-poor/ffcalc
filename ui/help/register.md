# Register

The starting point. Each row is one practice; click a row to open the Workbench.

## Using this screen

- **Add practice** — create a new one (you'll define demographics next).
- **Import CSV** — bulk-add from a Karo extract.
- **Edit** (per row) — change demographics or baseline.
- **Compare** — open the Comparison screen.
- The "@ 90% pass-through" offer column is illustrative only — based on a flat default, not your saved scenarios. Open a practice to model real numbers.
- **Variance vs current** is blank when the practice has no baseline saved. Edit the practice and add a Current PHO offer to populate it.

## Connections

- Click any row → **Workbench** for that practice.
- **Edit** → **Editor** for that practice.
