# Import & Export

Bring Karo roster extracts in; push results out. Files stay on this machine.

## Using this screen

The import is a four-stage flow: **Drop → Map → Preview → Done**.

- **Drop** the CSV (or click to choose).
- **Map** your CSV columns to FFCalc's fields. Required: practice name, type, all six age bands. Optional: female, Māori/Pacific, deprivation, CSC counts.
- **Preview** the parsed rows before committing.
- **Done** — practices land in the Register.

## Connections

- After import → **Register**, where the new practices appear.
