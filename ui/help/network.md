# Network

Portfolio-level rollup: every practice, one set of pass-through sliders, totals across the whole network.

## Using this screen

- Move sliders to set a single network-wide pass-through.
- The **Management services** toggle includes/excludes the per-enrolee management line that the PHO retains in full.
- Per-practice rows show how the network setting lands for each practice.
- Click any practice row → its Workbench, where you can model that practice individually.

## Network vs Comparison

- **Network** = one pass-through applied to *all practices*. For seeing portfolio totals.
- **Comparison** = different scenarios for *one or a few practices*, side-by-side. For deal-shaping.

## Connections

- Click a practice row → **Workbench**.
- For per-practice differentiation, save practice-level scenarios in the Workbench and use Comparison.
